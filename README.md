# Extended JS++ VSCode syntax

Original syntax was taken from [Onux/JS++](https://github.com/onux/jspp) repo.

## Installation:
Copy `jspp.plist` into `.vscode/extensions/onux.jspp-x.x.x/syntaxes/`.
